Add a fully-customizable store locator to your WordPress site.

== Description ==

Add a store locator to your WordPress powered site using Blipstar.

Embed your Blipstar-generated store locator using shortcode [blipstar id="###"] where id is your unique Blipstar account id.

Go to the [plugin info page](http://www.blipstar.com/blipstarplus/wordpress_store_locator_plugin.php?wordpress=true)

== Other Notes ==

Based on the iframe plugin (http://wordpress.org/extend/plugins/iframe/)

== Installation ==

1. install and activate the plugin on the Plugins page
2. click on the instructions link (shown at the top of the WordPress admin Dashboard)

==  Frequently Asked Questions ==

= How do I create and set up my store locator? =
Visit the [Blipstar](http://www.blipstar.com?wordpress=true) site

= What do I need to type into my page for the store locator to appear? =
Once you've created your store locator and are logged in to Blipstar, [click here](http://www.blipstar.com/blipstarplus/wordpress_store_locator_plugin_instructions.php) and it will tell you what the shortcode is.
